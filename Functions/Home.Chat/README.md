# Home.Chat – Azure Functions

## Running with Docker

```bash
docker compose up
```

## Ports

| Service | URL | Description |
|---|---|---|
| Azure Functions | `http://localhost:7071` | Function endpoints |
| Swagger UI | `http://localhost:7071/api/swagger/ui` | OpenAPI documentation |
| Seq | `http://localhost:8081` | Structured log viewer |
